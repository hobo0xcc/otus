# otus

otus is a toy functional programming language that has a GC and Type Inference, and use LLVM as backend.

# Build

```
make
```

# Run sample program

```
make
cd sample
```

### Hello, world!
**hello.ot**
```
make hello
./hello
```

### Floating point number
**float.ot**
```
make float
./float
```

### Garbage collection
**gc.ot**
```
make gc
./gc
```
