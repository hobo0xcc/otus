# otus

otus is a toy functional programming language that has a GC and Type Inference, and use LLVM as backend.

# Build

```
make
```

# Run example program

```
make
cd examples
```

### Hello, world!
**hello.ot**
```
make hello
./hello
```

### Floating point number
**float.ot**
```
make float
./float
```

### Garbage collection
**gc.ot**
```
make gc
./gc
```

### Fibinacci number
**fib.ot**
```
make fib
./fib
```

### FizzBuzz
**fizzbuzz.ot**
```
make fizzbuzz
./fizzbuzz
```

### Pointer
**pointer.ot**
```
make pointer
./pointer
```
